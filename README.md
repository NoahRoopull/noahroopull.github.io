# noahroopull.github.io
Static versions of a few web tools

Contains:
- index: a linktree to the tools provided
- imgtosprite: given an image, turn it into a sprite usable by Microsoft Makecode Arcade
- spritegallery: a large group of sprites that can be copied by clicking
- scriptrepository: a set of scripts that can be copied from; currently only contains ROBLOX Studio scripts
